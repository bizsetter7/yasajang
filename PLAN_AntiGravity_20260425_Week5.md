# Implementation Plan - Week 5 (P5 야사장 & P2 코코알바)

## 1. 개요
본 계획서는 P5 야사장의 구독 서비스 고도화 및 P2 코코알바와의 연동, 그리고 P2 상세 페이지 리디자인을 목표로 합니다.

## 2. 주요 작업 항목

### [P5 야사장] 구독 서비스 고도화
- **Pricing UI 개선**: 1/3/6/12개월 기간별 토글 UI 적용 및 할인율 계산 로직 구현.
- **결제 프로세스**: 결제 요청 시 '코코알바' 또는 '선수존' 플랫폼 선택 기능 추가.
- **입점 신청 로직**: `shops` 테이블을 `businesses` 테이블로 전환하고, 신청 시 `trial` 상태의 구독 정보를 자동 생성.
- **어드민 연동**: 결제 승인 시 P2의 업체 티어(`cocoalba_tier`)를 자동으로 동기화하는 로직 구현.

### [P2 코코알바] 데이터 정리 및 리디자인
- **데이터 정리**: 128개의 목업 광고 데이터(user_id 6fc68... 계열) 삭제 완료.
- **상세 페이지 리디자인**: '희야야' 스타일의 프리미엄 UI 적용 (`ShopDetailView` 신규 생성).
- **데이터 연동**: Supabase `shops` 테이블과 `businesses` 테이블을 조인하여 실제 업체 티어 정보를 반영하도록 수정.

## 3. 기술적 변경 사항
- **P5 DB 스키마 대응**: `subscriptions` 테이블에 `platform_choice` 필드, `businesses` 테이블에 `cocoalba_tier` 필드 활용 (데이터 정합성 확보).
- **Build 안정화**: Next.js 15+ 환경에서의 Webpack 빌드 최적화 및 `useSearchParams` 관련 Suspense 래핑 보완.

## 4. 검증 계획
- `npm run build`를 통한 정적 페이지 생성 및 런타임 에러 유무 확인.
- 플랫폼 선택 로직 및 어드민 승인 시 티어 업데이트 로직 검증.
