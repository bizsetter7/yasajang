# 야사장 플랫폼 Week 2 작업 완료 가이드

## 1. 어드민 심사 시스템 (`/admin/register-audit`)
관리자가 새로운 입점 신청을 검토하고 승인/거절할 수 있는 전용 대시보드를 구축했습니다.

### 주요 기능
- **실시간 목록**: Supabase 서버 연동을 통해 대기 중인 신청 건을 즉시 확인 가능.
- **심사 모달**: 
  - 신청 정보(대표자, 사업자번호, 연락처) 상세 검토.
  - 사업자등록증 원본 바로가기 제공.
  - 심사 의견 입력 및 원클릭 승인/거절 처리.

## 2. 텔레그램 실시간 알림 시스템
플랫폼 운영 상황을 모바일에서 실시간으로 파악할 수 있도록 텔레그램 봇을 연동했습니다.

### 알림 시점 (Trigger Events)
1. **신규 입점 신청**: 사용자가 입점 폼을 제출하는 즉시 발생.
2. **심사 결과 통보**: 관리자가 심사 대시보드에서 결과를 확정하는 즉시 발생.

### 구현 아키텍처
- **Secure API Proxy**: `TELEGRAM_BOT_TOKEN`의 클라이언트 노출을 방지하기 위해 `/api/notify` 서버 사이드 라우트를 통해 전송.

## 3. 안정성 및 품질 지표
- **TypeScript**: 100% 타입 안전성 확보 (누락 아이콘 및 프롭 오류 수정 완료).
- **Build**: `next build --webpack` 성공 (Windows 환경 검증 필).
- **UI/UX**: Zinc & Amber 컬러 팔레트 기반의 고대비 프리미엄 다크 모드 적용.
